Gym Genius is a cutting-edge, multi-platform mobile application designed to be your ultimate gym companion. Leveraging the power of **Google Gemini AI**, it provides personalized workout routines, nutrition plans, and expert coaching feedback—all wrapped in a stunning, high-performance dark-themed UI.

---

## 🌟 Key Features

### 🤖 AI Workout Generator
Say goodbye to generic plans. Generate custom routines based on your fitness goals, experience level, and available equipment.

### 🥗 Smart Meal Planner
Optimize your nutrition with AI-generated meal plans tailored to your macros and dietary preferences.

### 📊 Advanced Progress Analytics
Visualize your journey with interactive charts and comprehensive logs for workouts and weight.

### 🧠 AI Personal Trainer
Get instant analysis of your performance. Our AI trainer reviews your logs to provide actionable feedback for faster results.

## 🚀 Getting Started

### Prerequisites
- Flutter SDK (Latest Stable)
- Android Studio / VS Code
- A Firebase Project

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yashkoolwal27/Gym-Genius.git
   cd gym_genius_flutter
   ```

2. **Configure Firebase:**
   - Add `google-services.json` to `android/app/`
   - Add `GoogleService-Info.plist` to `ios/Runner/`
   - Run `flutterfire configure`

3. **Install dependencies:**
   ```bash
   flutter pub get
   ```

4. **Run the app:**
   ```bash
   flutter run
   ```
